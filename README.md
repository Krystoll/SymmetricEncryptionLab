# Лабораторная работа №2 – Симметричное шифрование

---

## Описание

Программа демонстрирует **симметричное шифрование** на Java.  

Функции:

- Генерация ключа и IV (вектора инициализации)  
- Выбор алгоритма (AES, DES)  
- Выбор режима работы (ECB, CBC)  
- Шифрование и расшифровка текста  
- Демонстрация различий при разных IV и режимах ECB/CBC
